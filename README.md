## ✨ Features

- ⏲️ `/settimer <seconds>` – Auto-delete after a delay  
- ❌ `/off` – Turn off message deletion  
- 📊 `/status` – View current settings  
- 🛡️ `/onlyadminon` – Delete non-admin messages only  
- ℹ️ `/info` – Get group/bot info  
- 🧠 `/help` – See all commands  
- 👮 **Admin-only restriction** for sensitive commands

---

## ⚙️ How to Use

### 1. Invite Bot to Group  
Add your bot to your group and promote it as **admin**.

### 2. Use Admin Commands  
Example:
```

/settimer 10

````
All messages will auto-delete after 10 seconds.

## 🔧 Self-Hosting Guide

Deploy this bot on platforms like [Koyeb](https://www.koyeb.com/), [Render](https://render.com), or Heroku-compatible services.

### 📁 Required Files
- `main.py`
- `Procfile`:
  ```bash
  web: python main.py
````

* `requirements.txt`:

  ```
  python-telegram-bot[webhooks]==20.7
  ```

### 🌱 .env File

```env
BOT_TOKEN=your_bot_token
WEBHOOK_URL=https://your-koyeb-app-name.koyeb.app
```
